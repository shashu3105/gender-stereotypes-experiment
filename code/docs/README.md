## Documentation

This folder contains supporting materials for the project
*Beliefs about Gender and Task Performance*.

### Contents

- `experimental-presentation.pdf`  
  Slides prepared for an academic course presentation, summarizing the
  experimental design, motivation, and illustrative analysis workflow.

### Important Note

The presentation is provided for context and communication purposes only.

- All figures and results shown are based on **synthetic data**
- The slides are **not intended as standalone empirical evidence**
